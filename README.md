# Operational-Analysis-of-Shared-Bikes-in-Shenzhen-During-the-Labor-Day-Holiday
### 深圳市共享单车劳动节期间运营分析

## 项目概述
- **数据处理**: Pandas, Numpy
- **数据库**: MySQL
- **可视化**: Matplotlib, Folium, Power BI
- **分析工具**: Jupyter Notebook, PyCharm 2025.1.2
## 核心发现
1. **运营洞察**: 识别早晚高峰模式， 订单量峰值出现在18:00左右
2. **用户行为**: 高频用户贡献了17.94%的订单量
3. **空间特征**: 发现10个热点区域，个供需失衡位置
4. **商业价值**: 通过调度优化预计可提升%的车辆利用率
